# arch-kt [![](https://jitpack.io/v/y1xian/arch-kt.svg)](https://jitpack.io/#y1xian/arch-kt)
## 使用方式
#### 1. 作为module导入
把arch-kt作为一个module导入你的工程。

#### 2. gradle依赖

```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}

dependencies {
    implementation 'com.github.y1xian:arch-kt:1.0.7'
}
```
